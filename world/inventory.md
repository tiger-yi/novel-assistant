# 道具与功法 (Inventory)

## 核心物品
*   **智脑·零号 (AI Zero)**
    *   **类型**：观想物 / 本命法宝（伪）
    *   **状态**：道OS 1.0 满载运行。
    *   **功能**：微观扫描、数据分析、情绪能量转化、高频震荡（物理干涉）、深度学习、多线程挂机。

*   **养魂玉 (Soul Jade)**
    *   **来源**：父亲方天正赠予。
    *   **功能**：温养精神力，是智脑启动的“备用电源”。
    *   **状态**：能量已耗尽（被智脑吸收）。

*   **聚气丹**
    *   **来源**：纳兰嫣退婚补偿。
    *   **功能**：练气期辅助修炼丹药。
    *   **状态**：未使用。

*   **爆血丹**
    *   **来源**：方厉赐予方烈。
    *   **功能**：短时间内提升一倍实力，副作用极大。
    *   **状态**：方烈持有。
