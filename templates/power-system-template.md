# 力量体系模板 (Power System)

## 1. 境界层级 (Realms)
- **淬体境**：锻体强血，夯实根基；重数：1–9
- **灵脉境**：开辟灵脉，灵气周天；重数：1–9
- **归元境**：归一真元，稳固道基；重数：1–9
- **化海境**：丹田化灵海，储能增幅；重数：1–9
- **登楼境**：神魂登楼，感应天地；重数：1–9
- **窥天境**：窥见天理，术法成势；重数：1–9
- **问道境**：问道证理，领域雏形；重数：1–9
- **证道境**：大道成章，开宗立派；重数：1–9

> 将本世界的实际境界名称与含义填入上方条目；重数可按需调整。

## 2. 突破规则 (Advancement Rules)
- **标准条件**：灵气/资源/悟性要求
- **失败后果**：受伤/退步/心魔
- **辅助途径**：丹药/法阵/秘境
- **冷却/恢复**：突破后需要的恢复时间

> 为每一大境界定义具体的突破条件与风险控制。

## 3. 技法与品阶 (Abilities & Grades)
- **品阶**：凡 / 黄 / 玄 / 地 / 天 / 圣 / 帝 / 禁
- **分类**：身法 / 剑法 / 体修 / 阵法 / 符术 / 神通 / 法宝
- **记录要求**：来源、消耗、冷却、适用境界 → 记录到 `world/inventory.md`

## 4. 战斗规则 (Combat Constraints)
- **越级上限**：不得跨越两大境界
- **单境越级代价**：寿元 / 虚弱 / 冷却 / 资源
- **法宝/阵法**：仅短时压制，战后记录虚弱与冷却窗口
- **描写规范**：招式名、光效、空间震荡、震惊侧写

## 5. 时间线与代价 (Timeline & Costs)
- **闭关/旅途/恢复**：在 `world/timeline.md` 标注具体时长
- **禁术使用**：设定虚弱期与冷却期，确保可检索

## 6. 一致性校验 (Consistency Checklist)
- 行为与当前境界匹配；杜绝战力崩坏
- 地图具“建议进入等级”；跨等级需给出代价或队友/阵法支持
- 掉落与收益与敌方境界、地形风险相称；同步到 `inventory.md`

## 7. 世界文件映射 (World File)
- **目标文件**：`world/power.md`
- **维护人**：作者/主控
- **更新时机**：新卷开篇 / 引入新体系 / 修订战斗规则
